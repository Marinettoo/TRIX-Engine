# TRIX - Threat Response & Intelligent eXecution

TRIX is a cybersecurity system that simulates being a honeypot, captures real attack data, analyzes behavior using risk scoring engine, and executes configurable response actions.

## Features

- SSH honeypot attack collection (Cowrie)
- Risk scoring engine (1-10)
- Configurable response policies
- REST API (FastAPI)
- Vulnerability scanning module

## Architecture
Cowrie -> Core Engine -> API -> Dashboard
